# 🚀 Fullstack User Management System

A full-stack application built using the MERN stack (MongoDB, Express, React, Node.js) that provides a scalable and secure system for managing users.

---

## 📌 Project Overview

This project demonstrates how to build a production-ready fullstack application with:

* RESTful API architecture
* Backend development with Node.js & Express
* Database integration using MongoDB
* Scalable and clean project structure
* Frontend integration (React - coming next)

---

## 🛠️ Tech Stack

### 🔹 Backend

* Node.js
* Express.js

### 🔹 Database

* MongoDB (Mongoose)

### 🔹 Tools

* Git & GitHub
* Postman

---

## ⚙️ Features

* ✅ Create a new user
* ✅ Get all users
* ✅ Get user by ID
* ✅ Update user details
* ✅ Delete a user
* ✅ Error handling
* 🔄 Frontend integration (in progress)

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/rajendra13542/fullstack-user-management.git
cd fullstack-user-management
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Start MongoDB

Make sure MongoDB is running locally:

```bash
mongod
```

---

### 4. Run the server

```bash
node server.js
```

Server runs at:
👉 http://localhost:3000

---

## 📬 API Endpoints

### ➤ Create User

POST /users

### ➤ Get All Users

GET /users

### ➤ Get Single User

GET /users/:id

### ➤ Update User

PUT /users/:id

### ➤ Delete User

DELETE /users/:id

---
